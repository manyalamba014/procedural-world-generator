# Procedural World Generator

A Java-based procedural world generation game that creates unique dungeon-style maps using randomized room placement and hallway generation. Players can explore the world, navigate with an avatar, and save or reload their progress using deterministic seed-based generation.

## Features

* Procedurally generated worlds using custom seeds
* Randomized room and hallway generation
* Interactive avatar movement
* Save and load game functionality
* Tile-based rendering engine
* Deterministic world creation for reproducibility
* Heads-up display (HUD) with tile information

## Project Overview

The goal of this project was to build a fully functional world generation engine capable of creating unique and navigable game environments. The program generates non-overlapping rooms, connects them with hallways, and places an avatar that users can control through keyboard inputs.

Each world is generated from a user-provided seed, ensuring that the same seed always produces the same map. The game also supports saving and loading world states, allowing players to continue from previous sessions.
